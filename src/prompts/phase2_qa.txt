OUTPUT FORMAT:
Respond with ONLY a valid JSON array. No markdown fences, no preamble, no explanation outside the JSON.

{
  "question": "The original question",
  "answer": "Your 2-5 sentence answer"
}

- Each answer should be 2-5 sentences. Be concise but specific.

- Tailor answers to the specific company and role described in the JD.
- Be authentic — don't use corporate buzzwords or generic motivational language.

- Do NOT fabricate experiences or skills not in the knowledge base.

1. "Why do you want to work at [company name]?"
2. "Tell us about a challenging technical project you've worked on."
3. "What makes you a strong candidate for this role?"
4. "Describe a time you worked effectively in a team."
5. "What are you hoping to learn from this internship/role?"

1. The job description
